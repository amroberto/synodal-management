# Synodal Management

A modern Synodal Management System built with Laravel 13, Livewire and Flux UI.

## About

Synodal Management is a web application designed to support the administrative management of synods, parishes, communities, members, leaderships and related processes.

This project is currently under active development.

## Built With

- PHP 8.5
- Laravel 13
- Livewire
- Flux UI
- Tailwind CSS
- MySQL

## Features

- Authentication
- Dashboard
- Community Management
- Parish Management
- Member Management
- Leadership Management
